# Capstone MERN Project

## Setup

### Server
cd server
npm install
cp .env.example .env
node server.js

### Client
cd client
npm install
cp .env.example .env
npm run dev

## Features
- Book management (CRUD)
- Task manager with add, update, delete
- Real-time chat with Socket.io
- React + Tailwind frontend
- Express + MongoDB backend
